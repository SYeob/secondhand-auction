# 테스트 시나리오

| ID | 시나리오 | 우선순위 |
|---|---|---|

| TS-LOGIN-002 | 잘못된 비밀번호로 로그인한다. | P1 |
| TS-LOGIN-003 | 비로그인 상태에서 보호된 화면에 접근한다. | P1 |
| TS-LOGIN-004 | 로그아웃 후 보호된 화면에 다시 접근한다. | P1 |

## 2. 상품 등록

| ID | 시나리오 | 우선순위 |
