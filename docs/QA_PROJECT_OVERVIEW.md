# Pa-Bi QA Automation Project

## 1. 프로젝트 목적

AI 기반으로 제작된 C2C 경매 서비스를 대상으로
요구사항 분석, 위험 기반 테스트 설계, UI/API 자동화,
CI 테스트 파이프라인 구축까지 수행하는 QA 포트폴리오 프로젝트입니다.

## 2. 테스트 대상

- 사용자 인증
- 상품 등록 및 관리
- 상품 조회 및 검색
- 실시간 입찰
- 경매 종료 및 낙찰
- 좋아요 및 마이페이지
- 개인정보 및 접근 권한
- 반응형 UI
- API 및 데이터 정합성

## 3. QA 수행 범위

- 요구사항 분석
- 위험 기반 테스트 전략
- 테스트 시나리오 및 상세 테스트 케이스
- 탐색적 테스트
- UI 자동화
- API 자동화
- 동시성 테스트
- 접근성 및 기본 성능 테스트
- CI 자동 실행
- 결함 관리 및 회귀 테스트

## 4. 제외 범위

- 대규모 운영 트래픽 검증
- 전문 침투 테스트
- Supabase 인프라 자체 운영 검증
- MCP 기능 전체 자동화

## 5. 테스트 기술

- Playwright
- TypeScript
- Postman / Newman
- GitHub Actions
- JMeter 또는 k6
- axe-core
- Docker
