# 테스트 전략 (Test Strategy)

## 1. 목적

본 프로젝트의 테스트는 Pa-Bi Auction 서비스의 핵심 기능인
회원 인증, 상품 등록, 입찰, 낙찰 처리 과정을 검증하여
서비스의 품질을 확보하는 것을 목적으로 한다.

또한 기능 검증뿐 아니라 보안, 회귀 테스트 및 탐색적 테스트를 수행하여
서비스 운영 시 발생 가능한 위험 요소를 사전에 식별하고 품질 리스크를 최소화한다.

---

## 2. 테스트 범위

### 포함 범위 (In Scope)

- 회원가입 및 로그인
- 상품 등록
- 상품 목록 조회
- 상품 상세 조회
- 입찰 기능
- 낙찰 처리
- 마이페이지
- 관심상품(좋아요)
- REST API 검증
- UI 자동화
- 보안 검증

### 제외 범위 (Out of Scope)

- 실제 결제 시스템
- SMS 발송
- 이메일 발송
- 외부 OAuth 로그인
- 대규모 부하 테스트
