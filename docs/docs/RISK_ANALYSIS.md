# Risk Analysis

## 1. 목적

Pa-Bi 경매 플랫폼의 주요 품질 위험을 식별하고, 발생 가능성과 영향도를 기준으로 테스트 우선순위를 결정합니다.

## 2. 산정 기준

- 발생 가능성: 1~5
- 영향도: 1~5
- 위험도 = 발생 가능성 × 영향도

| 위험도 | 등급   | 대응 기준                    |
| -----: | ------ | ---------------------------- |
|  15~25 | High   | 자동화 필수, 배포 전 검증    |
|   8~14 | Medium | 회귀 테스트 포함             |
|    1~7 | Low    | 탐색 테스트 또는 선택적 검증 |

| ID   | 위험 항목                                 | 가능성 | 영향도 | 위험도 | 등급   | 검증 방식     |
| ---- | ----------------------------------------- | -----: | -----: | -----: | ------ | ------------- |
| R-01 | 낮은 입찰가가 최고가로 저장됨             |      4 |      5 |     20 | High   | API, DB       |

## 4. 테스트 우선순위 반영

- High: 핵심 자동화 대상으로 선정
- Medium: 정기 회귀 테스트에 포함
- Low: 탐색 테스트 또는 선택적 자동화
